What inspired this repository was, while reading the book Programming: Principles and Practice Using C++ by the creator of C++ himself, Bjarne Stroustrup, I found that it was kind of boring to copy the code blocks given by the author manually into my PC. And even when opening it on a PDF and copying straight from the PDF,
I found that there were a lot of mistakes and a lot of punctuation missing and a lot of letters that are changed, basically giving your code 10+ problems, leading to just you deleting it and writing it from the start.

So I decided to create this repository to help everyone who faces the same problem as me. Keep in mind that this is based on the third edition of the book so if you are reading the first or second edition things may differ slightly.

there are changes between the editions so things will differ but what i am 100% sure of is that this repository will help you no matter what edition you are reading.

I did read this book with some past C++ knowledge, nothing too advanced, just nice tricks. So I tried implementing them and every major change I did to the files, I tried to put in a separate file so people who are going into this with no knowledge can still follow along.

So if you are someone who has C++ experience and is reading this book, this repository will be great for you. If not, you will still benefit massively because I have the initial code blocks of everything.

This also includes the solutions of the exercises, but keep in mind, a lot of the code blocks that were used as examples in the middle, I did not include. At least for now, I could add them later.

Also, I am active and will be accepting pull requests to this repository, given their accuracy, of course.

I did take a lot of liberties and added a lot of functionality to the files that was not explicitly requested by Stroustrup, especially error handling. You don't have to look at these optional files, but I would recommend readers to at least try to understand these concepts because you will need them later on and you will need them when writing real-life programs.

Also, you will notice that I added PPPheaders.h in every folder because the compiler wouldn't recognize it otherwise. I could have added a path, but it would have been different from my PC to yours. Were there better ways? Yes. Are they necessary? Not currently in my opinion. It's a small file.

Also, keep in mind that PPPheaders.h is not a part of "default" C++ per se, but it is intentional from Stroustrup to improve your experience and make the learning curve easier for beginners.

I haven't read the book fully yet, so I don't know if this is addressed properly, although I'm 98% sure it is.
